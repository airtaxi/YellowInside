# 옐로우 인사이드 (Yellow Inside)

메신저에서 디시콘을 사용할 수 있게 해주는 Windows 앱입니다.

![Showcase](.github/Assets/Showcase.gif)

## 요구 사항

- Windows 10 이상

[![Microsoft Store에서 다운로드](https://get.microsoft.com/images/en-US%20dark.svg)](https://apps.microsoft.com/detail/9PG8PCFFZ7BD)

## 주요 기능

- 메신저 채팅창에 디시콘 버튼 자동 부착
- 디시콘 이미지 검색 및 전송
- 시작 프로그램 등록 지원
